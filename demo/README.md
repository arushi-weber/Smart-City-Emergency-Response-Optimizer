# 🚑 Smart City Emergency Response Optimizer

## 📌 Project Description

This project helps to find the nearest emergency service like hospitals, police stations, or fire stations using a map.

It shows locations on a map and helps users quickly identify the closest help available. The goal is to improve response time during emergencies.

---

## ⚙️ Technologies Used

* Java (Backend)
* HTML, CSS, JavaScript (Frontend)
* Leaflet.js (Map)
* Maven (Project Management)

---

## 🗺️ Features

* Display map using Leaflet
* Show emergency locations
* Find nearest emergency service
* Simple and clean user interface

---

## ▶️ How to Run

1. Open the project in Eclipse
2. Run the Spring Boot application
3. Open browser and go to:
   http://localhost:9099/map.html

---

## 📁 Project Structure

* src/main/java → Backend code
* src/main/resources/static → Frontend files (HTML, CSS, JS)
* Emergency_Data.csv → Data file

---

## 📸 Output

The project shows a map with emergency locations and helps find the nearest service.
